# To-Do Web App

## Overview

This is a responsive To-Do Web Application developed as part of the Oasis Infobyte Web Development Internship.

The application allows users to manage daily tasks efficiently by adding, editing, completing, and deleting tasks. Completed tasks are automatically moved to a separate section for better task organization.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Separate Pending Tasks and Completed Tasks sections
* Date and time tracking
* Local Storage support (tasks remain after page refresh)
* Responsive and user-friendly interface
(Screenshots are avilable in images folder)

## Technologies Used

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in a web browser.
4. Start managing your tasks.
