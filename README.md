# OIBSIP - Oasis Infobyte Web Development & Designing Internship

## About

This repository contains all the tasks completed during my Oasis Infobyte Web Development & Designing Internship. The projects were developed using HTML, CSS, and JavaScript to strengthen my frontend development skills and gain practical experience in building interactive web applications.

---

## Tasks Completed

### Task 1 - Calculator

A responsive calculator capable of performing basic arithmetic operations.

**Features:**

* Addition, Subtraction, Multiplication, Division
* Responsive Design
* Interactive User Interface

**Technologies Used:**

* HTML
* CSS
* JavaScript

---

### Task 2 - Tribute Page

A tribute webpage dedicated to an inspirational personality.

**Features:**

* Attractive Layout
* Biography Section
* Achievements and Contributions
* Responsive Design

**Technologies Used:**

* HTML
* CSS

---

### Task 3 - To-Do Web App

A task management application for organizing daily activities.

**Features:**

* Add New Tasks
* Mark Tasks as Completed
* Separate Pending and Completed Task Sections
* Responsive Design

**Technologies Used:**

* HTML
* CSS
* JavaScript

---

### Task 4 - Login Authentication System

A simple authentication system that allows users to register, log in, access a protected dashboard, and log out.

**Features:**

* User Registration
* User Login
* Protected Dashboard Page
* Logout Functionality
* Local Storage Authentication
* Responsive Design

**Technologies Used:**

* HTML
* CSS
* JavaScript
* Local Storage
