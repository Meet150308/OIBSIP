# Calculator Web Application

## Overview

A simple and responsive calculator built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations through an interactive user interface and was developed as part of the Oasis Infobyte Internship Program (OIBSIP).

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Percentage (%)
* Decimal calculations
* Clear All (AC) button
* Delete Last Character (DEL) button
* Responsive and modern user interface
* Error handling for invalid expressions

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
(Also I have used and taken help of AI tools)

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.
4. Start performing calculations.
