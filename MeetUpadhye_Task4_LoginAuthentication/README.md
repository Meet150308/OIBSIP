# Login Authentication System

## Description

A simple Login Authentication System built using HTML, CSS, and JavaScript. This project allows users to register, log in, access a secured dashboard page, and log out. User credentials and login status are stored using Local Storage.

## Features

* User Registration
* User Login
* Authentication Validation
* Protected Dashboard Page
* Logout Functionality
* Responsive User Interface
* Local Storage for Data Persistence

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage
(I have used and taken help of AI tools)

## How to Run

1. Download or clone the project.
2. Open the project folder in VS Code.
3. Run `index.html` using Live Server.
4. Register a new user account.
5. Log in using the registered credentials.
6. Access the secured dashboard page.
7. Click Logout to end the session.
